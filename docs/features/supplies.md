# Feature: Supplies Requests

## Description

Employees can request supplies for their department.

Supervisors and administrators can approve requests.

---

## Actors

Employee
Supervisor
Admin

---

## Actions

Create supply request
View supply requests
Approve request
Reject request

---

## Data

item
quantity
department
status
requestDate

---

## Status

Pending
Approved
Rejected

---

## Rules

Employees can only create requests.

Supervisors can approve requests.

Admins have full access.
